# Application Blurb

I completed an independent customer-support response evaluation project focused on pairwise judgment of AI-generated answers. The work involved comparing Response A and Response B across 100 support tickets, selecting the preferred response, scoring each answer for accuracy and helpfulness, identifying hallucinated content, and documenting the reasoning behind each decision.

To make the project portfolio-ready, I created annotation guidelines, a QA checklist, a project summary, and a reflection document. This project demonstrates practical evaluation skills that are directly relevant to AI training and quality-review work, especially distinguishing responses that sound helpful from responses that are actually correct, grounded, and useful.

Key skills demonstrated:
- pairwise response evaluation
- hallucination detection
- rubric-based scoring
- QA documentation
- customer-support annotation reasoning
